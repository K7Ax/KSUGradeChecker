# راصد الدرجات 🎓

حيّاك الله! هذا برنامج بسيط يقعد يراقب درجاتك في بوابة جامعة الملك سعود (**Edugate**)،
وأول ما تطلع لك درجة جديدة... ما يقولك إياها على طول! لا، يبي يلعب باعصابك شوي 😄

يرسل لك في بوت **تيليجرام** ويخليك تخمّن درجتك: تضغط الأزرار (‎A+‎، ‎A‎، ‎B+‎ … ‎F‎)،
وإذا غلطت يخليك تعيد، وإذا صدقت يهنّيك 🎉. وإذا ما عندك صبر، في زر **«👀 ورّني الحين»**
يكشفها لك طوالي.

> ترى البرنامج يشتغل على جهازك انت، وبحسابك انت. بياناتك ما تطلع برّا جهازك أبد،
> ومعلومات دخولك ما تروح إلا لصفحة Edugate الرسمية، ولا أحد ثاني يشوفها.
> استخدمه على مسؤوليتك يا الغالي.

## وش يبيلك قبل لا تبدأ

- جهاز فيه **[Node.js](https://nodejs.org)** (نسخة 18 أو أجدد). إذا ما هو موجود، نزّله من الموقع الرسمي.
- حساب **تيليجرام**.
- حساب **Edugate** (أكيد عندك، انت طالب 😉).

## طريقة التشغيل (سهلة والله)

سوّيها بالترتيب وازهل:

**١) نزّل المشروع:** من فوق اضغط زر **Code** الأخضر ← **Download ZIP**، وبعدها فك الضغط.

**٢) افتح PowerShell جوّه مجلد المشروع، واكتب:**
```powershell
npm start
```
أول مرة بيقعد شوي يجهّز نفسه (ينزّل كل اللي يحتاجه ويسوّي لك ملف الإعدادات `.env`)،
وبعدها بيقولك «عبّي الملف». اصبر عليه، التنزيل أول مرة بس.

**٣) افتح ملف `.env` اللي ظهر لك، وعبّي ٣ خانات بس واحفظ:**
- `BOT_TOKEN` ← رمز البوت. تجيبه من [@BotFather](https://t.me/BotFather): ارسل له `/newbot`، يسوّي لك بوت ويعطيك رمز، انسخه وحطه هني.
- `EDUGATE_USERNAME` ← اسم المستخدم حقك في Edugate.
- `EDUGATE_PASSWORD` ← كلمة المرور حقتك.

> خانة `CHAT_ID` خلّها فاضية، لا تتعب نفسك — البوت بيربط نفسه معك لحاله بعد شوي.

**٤) اكتب مرة ثانية:**
```powershell
npm start
```
الحين البوت اشتغل. روح تيليجرام، افتح بوتك، وارسل له `/start`.
بيردّ عليك **«✅ تم الربط!»** ومن هنا وطالع بيراقب درجاتك **كل 10 دقايق** لحاله.
خلّه شغّال ورح ريح. 🎉

## 🔐 خصوصيتك وأمانك (اقرأها لو سمحت)

- ملف `.env` ومجلد `data` ما يترفعون أبد على GitHub — محطوطين بالاستثناء (`.gitignore`).
  يعني رمزك وكلمة مرورك وجلستك، كلها قاعدة بجهازك انت بس.
- معلومات دخولك تنكتب بس في صفحة Edugate الرسمية، ما تروح لأي مكان ثاني، ولا تنحفظ بأي محل.
- البوت ما يكلّم إلا انت. أي شخص ثاني يجي يضغط، يتجاهله على طول.
- لو الدخول فشل (يمكن كلمة مرور غلط)، البرنامج يوقف لحاله ما يكرر المحاولة — عشان لا
  يقفل حسابك — ويرسل لك يقولك صلّح كلمة المرور.
- نصيحة: إذا كنت قبل عطيت رمز البوت لأي أحد، جدّده من [@BotFather](https://t.me/BotFather) بأمر `/revoke`.

**ولو نشرت المشروع على GitHub:** قبل لا ترفع، تأكد إن أمر `git status` ما يطلّع لك
`.env` ولا `data`. لو طلعوا، وقّف على طول وراجع ملف `.gitignore`.

## لو صار شي غلط

- **يقولك الجلسة انتهت؟** عبّي اسم المستخدم وكلمة المرور في `.env` وبيدخل لحاله. أو سجّل دخول يدوي بنفسك: `npm run login`.
- **الدخول ما ضبط؟** غالباً كلمة المرور غلط — صلّحها في `.env` وشغّل من جديد.
- **ما طلعت الدرجات؟** جرّب `npm run scrape` يطبع لك اللي قراه، وتأكد إنه يوصل لجدول الدرجات.

## للي يبي يفهم الكود (اختياري)

| الملف | وش يسوي |
|------|---------|
| `src/index.js` | يشغّل البوت وحلقة الفحص كل ١٠ دقايق. |
| `src/scraper.js` | يدخل Edugate ويسحب الدرجات من صفحة النتائج. |
| `src/bot.js` | بوت تيليجرام: يرسل اللعبة ويعالج تخميناتك وزر الكشف. |
| `src/checker.js` | يفحص، يقارن، ويبدأ لعبة لكل درجة جديدة. |
| `src/store.js` | يحفظ الدرجات اللي شافها واللعبات في `data/state.json`. |

وبالتوفيق في درجاتك يا بطل 💪
