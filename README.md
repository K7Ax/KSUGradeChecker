# مدقّق الدرجات — KSU Grade Checker 🎓

أداة تراقب بوابة **إيدوجيت** بجامعة الملك سعود وتنتظر ظهور **نتائج المقررات**.
بمجرد رصد درجة جديدة، يبدأ بوت **تيليجرام** معك **لعبة تخمين**: تظهر أزرار الدرجات
(‎A+‎، ‎A‎، ‎B+‎ … ‎F‎)، وكل تخمين خاطئ يتيح لك إعادة المحاولة، والتخمين الصحيح يكشف
درجتك مع احتفال 🎉، وهناك زر **«👀 أرني الآن!»** يكشف الدرجة مباشرةً.

> ⚠️ أداة شخصية تعمل **بحسابك أنت** على جهازك. بياناتك لا تغادر جهازك إطلاقًا،
> وتُرسل بيانات الدخول **حصراً** لصفحة دخول إيدوجيت الرسمية. استخدامها على مسؤوليتك.

---

## المتطلبات

- **Node.js** الإصدار 18 أو أحدث.
- حساب **تيليجرام**.
- حساب **إيدوجيت** بجامعة الملك سعود.

## التنصيب

```powershell
npm install
npx playwright install chromium
```

## الإعداد خطوة بخطوة (سهل)

1. **أنشئ بوت تيليجرام:** راسل [@BotFather](https://t.me/BotFather)، أرسل `/newbot`،
   وانسخ **رمز البوت** (`BOT_TOKEN`).
2. **شغّل البرنامج أول مرة** — سيُنشئ لك ملف الإعدادات `.env` تلقائيًا ويطلب منك تعبئته:
   ```powershell
   npm start
   ```
3. **افتح ملف `.env`** (سيظهر في مجلد المشروع) واملأ **٣ حقول فقط**، ثم احفظ:
   - `BOT_TOKEN` — رمز البوت من الخطوة ١.
   - `EDUGATE_USERNAME` — اسم المستخدم في إيدوجيت.
   - `EDUGATE_PASSWORD` — كلمة مرور إيدوجيت.

   > لا حاجة لتعبئة `CHAT_ID` — يربط البوت نفسه بك تلقائيًا في الخطوة التالية.
4. **شغّل مرة أخرى ثم اربط البوت:**
   ```powershell
   npm start
   ```
   افتح تيليجرام، ادخل على بوتك، وأرسل `/start`. سيردّ **«✅ تم الربط!»** ويبدأ
   فحص إيدوجيت **كل ١٠ دقائق** تلقائيًا. اتركه يعمل — هذا كل شيء. 🎉

## الأوامر

| الأمر | الوظيفة |
|------|---------|
| `npm start` | يشغّل البوت وحلقة الفحص الدورية. |
| `npm run login` | تسجيل دخول يدوي في متصفح (بديل عن الدخول التلقائي). |
| `npm run scrape` | فحص واحد يطبع النتائج المُستخرجة (لاختبار الاستخراج). |

## 🔐 الأمان (مهم)

- ملف **`.env`** ومجلد **`data/`** **لا يُرفعان أبدًا** إلى GitHub — كلاهما مُستثنى في
  `.gitignore`. بياناتك (الرمز، كلمة المرور، الجلسة) تبقى على جهازك فقط.
- بيانات دخول إيدوجيت تُستخدم **حصراً** لتسجيل الدخول في صفحة إيدوجيت الرسمية،
  ولا تُرسل لأي طرف ثالث ولا تُسجَّل في أي مكان.
- البوت يتفاعل **فقط** مع `CHAT_ID` الخاص بك؛ أي محادثة أخرى تُتجاهل.
- عند فشل تسجيل الدخول، يتوقف البوت عن إعادة المحاولة تلقائيًا **لحماية حسابك من
  الإيقاف**، ويُنبّهك على تيليجرام لتصحيح كلمة المرور.
- **جدّد رمز البوت من [@BotFather](https://t.me/BotFather)** (الأمر `/revoke`) إن سبق
  أن شاركت الرمز مع أي شخص، قبل أن تجعل المستودع عامًّا.

### قائمة تحقق قبل النشر على GitHub

قبل أي `git push`، تأكد أن الأمر التالي **لا يُظهر** `.env` أو `data/` إطلاقًا:

```powershell
git status
```

إن ظهر أيٌّ منهما، أوقف العملية فورًا وتأكد من `.gitignore`.

## استكشاف الأخطاء

- **انتهاء الجلسة:** إن لم تكن بيانات الدخول مضبوطة، سيطلب منك البوت تشغيل
  `npm run login`. مع ضبط البيانات يُعيد الدخول تلقائيًا.
- **فشل تسجيل الدخول:** غالبًا كلمة مرور خاطئة — صحّحها في `.env` وأعد التشغيل.
- **لم تُستخرج النتائج:** شغّل `npm run scrape` للتأكد من قراءة جدول الدرجات.

## كيف يعمل

| الملف | الدور |
|------|------|
| `src/login.js` | تسجيل دخول يدوي مرة واحدة وحفظ الجلسة في `data/auth.json`. |
| `src/scraper.js` | تسجيل دخول تلقائي + استخراج الدرجات من صفحة النتائج. |
| `src/store.js` | حفظ النتائج المرصودة والألعاب النشطة في `data/state.json`. |
| `src/checker.js` | يفحص ويقارن ويبدأ لعبة لكل درجة جديدة. |
| `src/bot.js` | بوت grammY: يرسل الألعاب ويعالج التخمينات وزر الكشف. |
| `src/index.js` | يشغّل البوت وحلقة الفحص كل ١٠ دقائق. |
